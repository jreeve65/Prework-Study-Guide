HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The head although hidden from the user's view is important because it contains information related to Search Engine Optimaization letting your website reach more people
  • A common tag used in the head is "Meta". Commands commonly used within the "Meta" tag are "name", "charset", "content", and "http-equiv".
  • The body element contains the content that is visible to the user
  • Common things put in the body are things such as Hyper links, images, and text
  • Useful commands for body are, "ul" for unordered lists, and "li" to display a listed item

CSS

  • A margin indicates how much space we want around the outside of an element.
  • there are several ways to implement css code in your program you can do it inline within your html file, you can use an internal style sheet or you can use an external style sheet. However, most cases you will use an external style sheet for cleaner code then link it to your html file.
  • A padding indicates how much space we want around the content inside an element.
  • You can add color using a semantic statement such as the name of the color or by using the hashtag symbol followed by its hexidecimal reference.
  • In CSS you code using a selector or stating an aspect of the code you want to effect designateing a property ie a color, text alignment etc. and then assgning that property a value ie what color do you want to use.
  • You can even assign multiple selectors in the same line of code eliminating the need to repeat already written code using a comma after the first selector followed by the second

GIT

  • PWD lets you know what directory you are in.
  • CD stands for change directory and allows you to navigate different files on your machine
  • MKDIR stands for make directory and it allows you to create a folder on your machine inside your current directory
  • GIT STATUS lets you know what branch of a project/repository you are currently working on
  • GIT CHECKOUT moves you to a existing branch with in a project/repository. You can also issue the command GIT CHECKOUT -B to create a new branch and change it to your current branch.

HTML

The first step to building starts with code using Hyper Text Markdown Language Commonly known as "HTML"

The structure of a HTML file is as follows

  • A head
  • A body
  • A footer

There are several other tags used to structure these individual parts but these three tags are basically the soul of the webpage or web application

The Head

  • The head element contains information about the webpage.
  • The head although hidden from the user's view is important because it contains information related to Search Engine Optimaization letting your website reach more people
  • A common tag used in the head is "Meta". Commands commonly used within the "Meta" tag are "name", "charset", "content", and "http-equiv".

The Body

  • The body element contains the content that is visible to the user
  • Common things put in the body are things such as Hyper links, images, and text
  • Useful commands for body are, "ul" for unordered lists, and "li" to display a listed item

GIT

Git is a method of version control development. Version control allows web and software developers collaberate by testing and adding/modifiying code in tandum whilst leaving the live service/feature unaffected until the changes are ready to be implimented.

Version control will also allow you to roll back to an earlier version of the code if the new one has a problematic bug that wasn't caught in testing

You can use git via the command line. a popular command line program is git bash some useful git bash commands are as follows

  • PWD lets you know what directory you are in.
  • CD stands for change directory and allows you to navigate different files on your machine
  • MKDIR stands for make directory and it allows you to create a folder on your machine inside your current directory
  • GIT STATUS lets you know what branch of a project/repository you are currently working on
  • GIT CHECKOUT moves you to a existing branch with in a project/repository. You can also issue the command GIT CHECKOUT -B to create a new branch and change it to your current branch.
